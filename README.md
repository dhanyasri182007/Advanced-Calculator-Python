# Advanced Calculator (Python)

This project simulates an advanced calculator that performs basic arithmetic operations and handles errors efficiently.

## Features

- Addition  
- Subtraction  
- Multiplication  
- Division  
- Handles Division by Zero Error  
- User-friendly Input System  
- Continuous Calculation Option  

## Concepts Used

- Functions (Modular Programming)  
- Exception Handling  
- Conditional Statements  
- Loops (Menu-driven execution)  
- Input Validation
